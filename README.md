# Veronzo Stone

Премиальный адаптивный одностраничный сайт для компании, работающей с натуральным камнем, элитными дверями и лифтовыми решениями.

## Стек

- HTML5
- CSS3 (переменные, Grid, Flexbox, без препроцессоров)
- Vanilla JavaScript (без фреймворков и сборщиков)

Проект статический: не требует сборки, backend или базы данных — достаточно открыть `index.html` или раздать папку любым статическим сервером.

## Возможности

- Адаптивная вёрстка от 320px до 1920px без горизонтального скролла
- Доступность: семантическая разметка, `:focus-visible`, `aria-live`/`aria-invalid` в форме, поддержка `prefers-reduced-motion`
- Лёгкие CSS/JS-микроанимации: плавное появление секций при скролле (`IntersectionObserver`), анимация hero, счётчик статистики
- Локальные изображения в AVIF/WebP с фолбэком через `<picture>`, точные `width`/`height` для предотвращения layout shift
- Базовый SEO: уникальный `title`, `meta description`, Open Graph, `theme-color`
- Клиентская валидация формы обратной связи (демо-режим — реальная отправка на сервер не подключена)

## Как запустить локально

Проект не требует установки зависимостей. Достаточно раздать корневую папку любым статическим сервером, например:

```bash
python3 -m http.server 8000
```

и открыть `http://localhost:8000` в браузере. Открытие `index.html` напрямую через `file://` тоже работает, но локальный сервер рекомендуется для корректной работы относительных путей и `<picture>`.

## Статус проекта

Пройдены этапы: frontend-аудит, доступность, адаптивность, базовый SEO, микроанимации, локальные оптимизированные изображения, favicon. Форма обратной связи работает только в демо-режиме — подключение к серверу является следующим шагом.
